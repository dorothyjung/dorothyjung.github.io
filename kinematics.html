Assignment 4

Inverse Kinematics Solver

Dorothy Jung (cs184-eu), Gautam Gunjala (cs184-el)

From project directory, type 'make' and then './as4' to run. Press 'n' to stop and start animation. Press 'j' for Lemniscate, 'k' for Flower, 'l' for Limacon. Dynamically changes goal path. Use arrow keys to rotate and spacebar to exit.

We used an arm with 4 segments of lengths of 5.0, 4.0, 3.0, and 2.0. Our arm tries to reach a goal using an inverse kinematics solver by calculating the angles between segments. We calculated the pseudo-inverse of the Jacobian matrix for each joint by using the Eigen SVD solver. Results that gave an out-of-reach goal were handled by scaling the vector toward the goal so that the arm was completely taut and extended.
